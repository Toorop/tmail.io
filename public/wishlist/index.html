Wishlist

Thu, Feb 26, 2015

Cette page recense les fonctionnalités que vous souhaitez voir implémentées dans tmail. Si vous avez une fonctionnalité que vous aimeriez voir intégrée:

  • Vérifiez qu’elle n’est pas dans la liste ci dessous.
  • Créez une nouvelle discussion sur le groupe tmail-dev pour la présenter et nous permettre d’en discuter.

En cours de discussion

  • Alias et listes de discussions
  • Restrictions sur l’expéditeur déclaré d’un mail en fonction de l’utilisateur SMTP
  • Pouvoir rediriger les logs ailleurs que vers stdout
